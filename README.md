### 대구대학교 흡연장 알리미 어플 (Daegu_University-Smoking_App)
### 개발 일자: 2018.03 ~ 2018. 06
### 개발 언어: Java
### 개발 프로그램: Android Studio


### 주요기능
- 내가 있는 위치 찾기 가능
- 마커 찍기
- google API사용
- 지도 확대 축소 가능
- 구글 지도 사용
- 대구대학교 흡연장 위치 

### 화면 구성
- 첫 화면
- 지도 화면
- 마커 클릭 시 이미지
